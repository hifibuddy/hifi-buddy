HiFi Buddy is now a real macOS app — drag-to-Applications, no Terminal needed.

## Highlights

- 🎉 **Native macOS app** (`HiFi-Buddy.dmg`) — drop into `/Applications`, right-click → Open the first time, then it just works. Menu-bar icon, no Dock clutter, no Python install required.
- 🟢 **Connect to Spotify in Settings** — a real "Connect" button next to your Client ID. PKCE redirect or Client Credentials, your call.
- 🎚 **Reference Library plays Spotify clips in-app** — same player bar as Plex/local clips, with a scrubber that tracks the clip segment (not the whole song).
- 🔁 **Pause/resume on the playline** — the ▶/⏸ button next to the progress bar is now a real toggle, not just a state indicator.
- 🌊 **Brand polish** — dashboard hero, welcome card, top bar all share the same wave + gradient treatment.

## Fixes

- ABX "Cannot perform slice on detached ArrayBuffer" decoder error → real codec error now surfaces instead.
- Lesson generator no longer rejects whole drafts when the AI omits the optional equipment block.
- Default port aligned to **8090** across the codebase — matches the most common Spotify Dev app redirect URI.
- Player bar resized so the "REFERENCE CLIP · …" chip fits inside.

## Install

Easiest: download `HiFi-Buddy.dmg` from the assets below.

> First launch: macOS Gatekeeper will refuse to open an unsigned app. **Right-click → Open**, then confirm "Open Anyway" in *System Settings → Privacy & Security* if prompted. After this one-time confirmation, double-clicking works normally.

Or run from source: `git clone https://github.com/hifibuddy/hifi-buddy && cd hifi-buddy/hifi-buddy-app && python3 server.py`.
